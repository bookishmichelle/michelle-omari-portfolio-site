Usv Gpt

USV GPT is a document ingestion AI program built using Ollama 3.1 and integrated into USV's existing software platform. Students can upload documents and receive AI-generated responses and summaries. While working as a team, I assisted in building the system, training it, and I individually worked on the UI to make it fit the organization's theme.

Obscurity

Obscurity is a 2D side scroller designed specifically for users with disabilities. I worked on this project as both a project manager and an engineer while simultaneously leading another team for a different project. We used agile methodology to guide our development process.
